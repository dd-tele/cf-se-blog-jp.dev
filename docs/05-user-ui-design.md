# 05 - ユーザー UI 設計

## 1. 概要

ユーザー UI は2つの大きなエリアで構成:
- **Public Blog（公開エリア）** — 誰でもアクセス可能。記事閲覧、検索、AI チャット
- **User Portal（認証エリア）** — ログインユーザー専用。記事執筆、プロフィール管理

**エディタ機能:**

| 機能 | 説明 |
|---|---|
| **AI 下書き生成** | テンプレート入力から Workers AI が完全な Markdown 下書きを自動生成 |
| **完全自由編集** | AI 生成後の下書きはすべて自由に書き換え・削除・追加可能 |
| **Markdown エディタ** | textarea ベース。Markdown 記法で直接記述 |
| **Markdown プレビュー** | marked + DOMPurify でリアルタイムプレビュー |
| **画像アップロード** | ドラッグ&ドロップ → R2 直接アップロード → CDN URL 自動挿入 |
| **画像自動配置** | テンプレート入力時のアップロード画像を AI が適切な位置に配置 |
| **自動保存** | 10秒ごとに KV に下書き保存。オフライン復帰時もデータ保持 |
| **AI 再生成** | 入力内容を変更して AI に再度下書きを生成させることが可能 |
| **AI アシスタント** | 要約生成、SEO 提案、タグ提案、文章改善 |
| **ワンクリック公開** | 下書き保存後、いつでも公開ボタンで即時公開 |
| **OGP プレビュー** | SNS シェア時のプレビュー確認 |
| **バージョン履歴** | 主要な保存ポイントの履歴 |
