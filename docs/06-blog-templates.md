# 06 - ブログテンプレート設計

## 1. テンプレート設計方針

**目的:** アイディアがあってもなくても、すぐにブログ執筆を開始できるガイド付きテンプレートを提供する。

**設計原則:**
- **構造化入力フォーム** — 必須項目・追記・画像を入力するだけで AI が下書きを生成
- **AI 下書き自動生成** — Workers AI が入力情報から完全な Markdown ブログ記事を作成
- **自由編集可能** — 生成された下書きはリッチエディタで 100% 自由に書き換え可能
- **難易度レベル** — 初心者・中級者・上級者に対応（入力項目の数で調整）
- **画像自動配置** — アップロード画像と説明を AI が記事の適切な位置に配置

## 3. テンプレート詳細（構造化入力フィールド + AI 下書き生成）

### 設計方針

従来のマークダウンテンプレート（穴埋め式）ではなく、**構造化入力フォーム → AI 下書き自動生成** 方式を採用。

```
ユーザーの流れ:
1. テンプレートを選択
2. 必須項目・追記・画像をフォームに入力
3. [AI で下書きを生成] をクリック
4. Workers AI が完全な Markdown ブログ記事を生成
5. リッチエディタで自由に編集・加筆・削除
6. 公開 or レビュー提出
```

**メリット:**
- アイデアが少なくても、必須項目を埋めるだけで記事の下書きが完成する
- 画像と説明を入力すれば、AI が適切な位置に配置する
- 生成された下書きは 100% 自由に書き換え可能

### 3.1 T-APP-01: Application Services 設定ガイド

**入力フィールド定義:**

| # | フィールド名 | 種類 | 必須 | 説明 |
|---|---|---|---|---|
| 1 | サービス名 | テキスト | * | 扱う Cloudflare サービス（例: WAF, CDN, Load Balancing） |
| 2 | 目的・ユースケース | テキストエリア | * | 何を実現するか（例: WordPress を WAF で保護する） |
| 3 | 対象読者 | テキスト | * | 想定する読者レベル |
| 4 | 設定手順（箇条書き） | テキストエリア | * | 主な設定ステップをざっくり列挙 |
| 5 | 使用する Cloudflare サービス一覧 | タグ選択 | * | 関連サービスを複数選択 |
| 6 | 前提条件・プラン | テキスト | | 必要なプランやアカウント要件 |
| 7 | トラブルシューティング | テキストエリア | | よくある問題と解決策 |
| 8 | 補足・特記事項 | テキストエリア | | 追加で記事に含めたい内容 |
| 9 | 参考 URL | URL リスト | | 公式ドキュメント等のリンク |
| 10 | 画像 + 説明 | 画像アップロード (複数) | | 構成図、スクリーンショット等。各画像に説明文を付与 |

**AI が生成する記事の構成:**
`はじめに → 構成概要(図) → 使用サービス一覧 → 前提条件 → 設定手順(Step形式) → 動作確認 → トラブルシューティング → まとめ → 参考リンク`
